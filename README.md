# 🛒 Supermarket Sales Analysis & Interactive Dashboard

## 📌 Project Overview

This project analyzes supermarket sales data using SQL and visualizes
the insights through an interactive dashboard built with HTML, CSS, and
JavaScript.

The goal is to transform raw transactional data into meaningful business
insights, including sales performance, customer behavior, and time-based
trends.

---

## 🧠 Business Questions

- What is the total number of orders?
- How does sales performance vary by branch?
- Which product lines generate the highest revenue?
- How do different customer types behave across product lines?
- Which hours have higher-than-average orders in the top-performing
  month?
- How does sales accumulate over time (monthly trend)?

---

## 🛠️ Tech Stack

- MySQL
- SQL (CTE, Subquery, Window Function)
- HTML / CSS / JavaScript
- Chart.js
- PapaParse

---

## 📊 Dashboard Features

- Upload CSV data from MySQL
- Filter by Branch and Customer Type
- KPI metrics (Sales, Orders, Top Month)
- Interactive charts

---

## 🚀 How to Use

1.  Run SQL queries in MySQL
2.  Export data to CSV
3.  Open index.html
4.  Upload CSV
